Sobre DEVo

Conoce el origen, el stack de desarrollo y la evolución de este repositorio como un proyecto de experiencia significativa.

Desarrollador Web & Apasionado de la Computación

Procesando código nativo, un qubit a la vez.

Hola, soy DEVo, creador de Quantum Hub. Este espacio nació con la meta de explorar temas complejos de la informática como lo es la computación cuántica, y presentarlos de forma clara y accesible. A través del bootcamp de JavaScript de JSCamp, optimicé este proyecto de principio a fin utilizando tecnologías limpias y nativas, con el objetivo de demostrar que el diseño visual sobresaliente y la arquitectura de código semántica no requieren de frameworks pesados para impresionar.

Evolución del Proyecto

Fase Inicial - Primer Prototipo

Estructura Base en HTML & CSS Nativo

Creación del artículo extenso de investigación sobre qubits, entrelazamiento cuántico e implicaciones de ciberseguridad. Se implementaron los primeros iframes de videos y un layout básico usando un flexbox clásico.

Optimización - Bootcamp JSCamp

Refactorización Semántica & SEO

Mejora completa de la semántica HTML5, adición de metadatos Open Graph, Schema.org para enriquecer los resultados de búsqueda, y optimización de accesibilidad mediante roles y atributos ARIA.

Fase Final - Interactividad y UI Premium

Módulos JS y Canvas Particles

Integración de ES Modules independientes, la API Intersection Observer para la navegación dinámica de secciones y la barra de lectura, un tema oscuro/claro persistido en localStorage y simulación cuántica interactiva en Canvas.

Tecnologías Utilizadas

HTML5 Semántico CSS Custom Properties Flexbox & Grid Layout JavaScript ES6+ Intersection Observer HTML5 Canvas API ES Modules (Modular JS) Web Accessibility (a11y) SEO & Structured Data
